# CodSoft Data Science Internship

## Overview
This repository contains the tasks completed during the **CodSoft Data Science Internship**. The projects focus on data analysis, machine learning, data preprocessing, model training, and prediction using Python and popular data science libraries.

## Internship Tasks
### Task 1: Titanic Survival Prediction
**Objective:** Predict whether a passenger survived the Titanic disaster using machine learning techniques.
**Files:**
* Titanic-Dataset.csv
* task1_titanicds_prediction.ipynb
* README.md

### Task 2: Movie Rating Prediction
**Objective:** Predict movie ratings based on various movie attributes.
**Files:**
* IMDb movie dataset.csv
* Task2_MoviePrediction.ipynb
* README.md

### Task 3: Credit Card Fraud Detection
**Objective:** Detect fraudulent credit card transactions using machine learning algorithms.
**Files:**
* task3_CreditCardFraudDetection.ipynb
* README.md
